# Part 4: AI Solution Design

This folder contains the AI business solution design for a telecom customer churn prediction problem.

## Contents

- `solution_report.md`: Detailed business problem definition, AI task classification, data requirements, model recommendation, evaluation plan, responsible AI considerations, and final solution summary.
- `diagrams/solution_architecture.png`: Architecture diagram illustrating the end-to-end AI solution flow.
- `ai_usecase_reference_catalog.csv`: Reference use case catalog for selecting the domain and AI task.
- `business_kpi_sample.csv`: Sample business KPIs for linking the AI solution to measurable impact.

## How to use

1. Open `solution_report.md` to review the solution design.
2. View `diagrams/solution_architecture.png` for the architecture flow.
3. Optionally use the reference CSV files to support the business and KPI rationale.
